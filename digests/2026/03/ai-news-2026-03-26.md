# AI News Digest - 2026-03-26

## Claude Code / Anthropic

### Claude Code v2.1.83リリース（3/25）
エンタープライズ向け機能が充実。`managed-settings.d/` ドロップインディレクトリでチーム単位のポリシー管理が可能に。`CwdChanged`/`FileChanged` フックイベント追加でdirenv連携に対応。`sandbox.failIfUnavailable` 設定、`CLAUDE_CODE_SUBPROCESS_ENV_SCRUB=1` によるサブプロセスの認証情報除去など、セキュリティ面も強化。画像ペースト時に `[Image #N]` チップが挿入される改善も。
- https://code.claude.com/docs/en/changelog

### Claude Auto Mode — リサーチプレビュー開始（3/24-25）
Claude Codeに「Auto Mode」が追加。低リスクの権限判断をAI安全性分類器が自動で行い、開発者の承認操作を削減する。Team プランで利用可能、Enterprise/APIは近日対応予定。
- https://www.buildfastwithai.com/blogs/claude-code-auto-mode-2026

### Claude Computer Use — フルエージェント化（3/24）
Claude Desktopからアプリ操作・ブラウザ操作・スプレッドシート入力などをエージェントとして実行可能に。スマートフォンからタスクを送信し、PCで自動実行する使い方も。アプリアクセス時は毎回許可を求めるセーフガード付き。
- https://www.cnbc.com/2026/03/24/anthropic-claude-ai-agent-use-computer-finish-tasks.html

## OpenAI

### レガシー Deep Research モード廃止（3/26）
旧版 Deep Research モードが本日付で削除。現行の Deep Research 体験は継続、過去の会話・結果は引き続きアクセス可能。

### GPT-5.4 mini ロールアウト
ChatGPTでGPT-5.4 miniが展開中。Free/Goユーザーは「Thinking」機能経由で利用可能。他プランではGPT-5.4 Thinkingのレートリミット時フォールバックとして提供。

### ChatGPT Library 機能追加
アップロード・作成したファイルを自動保存し、チャット間で再利用できる「Library」機能がリリース。ファイル検索やサイドバーブラウジングに対応。
- https://releasebot.io/updates/openai

### Stripe Minions — 自律コーディングエージェント
Stripeのエンジニアチームが「Minions」を公開。Slack・バグレポート・機能リクエストからタスクを受け取り、週1,300以上のPRを自動生成する自律型コーディングエージェント。
