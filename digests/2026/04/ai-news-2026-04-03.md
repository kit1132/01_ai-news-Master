# AI News Digest - 2026-04-03

## Cursor 3.0 リリース — エージェント中心の新インターフェース

Cursor が 3.0 をリリース（4/2）。VS Code フォークから脱却したゼロベースの新UIを導入。複数エージェントの並列実行、マルチリポジトリレイアウト、ローカル/クラウドエージェントのシームレス切替が可能に。Design Mode でブラウザ上のUI要素を直接アノテーションし、エージェントに精密なフィードバックを送れる。`/worktree`（git worktree で隔離実行）、`/best-of-n`（複数モデルで同じタスクを並列実行・比較）コマンドも追加。

- https://cursor.com/blog/cursor-3
- https://cursor.com/changelog

## GPT-4o 完全廃止 — 全プランから削除完了

4/3、Business/Enterprise/Edu の Custom GPTs での延長アクセスが終了し、GPT-4o が全プランから完全に廃止された。API は別スケジュールで引き続き利用可能。GPT-5.2 以降への移行が完了。

- https://help.openai.com/en/articles/20001051-retiring-gpt-4o-and-other-chatgpt-models

## Gemini 3 Pro ロールアウト開始 — Gemini アプリのデフォルトモデルに

Gemini 3 Pro が全ユーザーに展開開始。テキスト・画像・音声・動画のマルチモーダル推論が大幅改善。Deep Research でユーザーファイル/画像のアップロードが可能に。Canvas でインタラクティブ可視化・クイズ生成にも対応。Gems の共有機能も追加。日本を含む4カ国の18歳以上の学生に7月まで無料アップグレードを提供。

- https://gemini.google/release-notes/

## Devin — PR Resuming・リアルタイムターミナル出力

Devin が PR Resuming を追加（4/1）。別セッションで作成された既存PRを引き継いで作業可能に。ターミナル出力のリアルタイムストリーミング、Reviewの「残りレビュー行数」カウンター追加。GitHub Enterprise Server の org レベル App 登録にも対応。設定ページを統一レイアウトにリデザイン。

- https://docs.devin.ai/release-notes/overview

- https://workspaceupdates.googleblog.com/

## OpenAI — ChatGPT 機能アップデート

- **ライブラリ機能**: アップロード・作成したファイルを自動保存し、チャットで再利用可能に
- **ショッピング改善**: 画像ベースの類似商品検索、価格・レビュー・機能の並列比較
- **Box/Notion/Linear/Dropbox 統合更新**: 書き込み機能を含む新アクション追加

- https://help.openai.com/en/articles/6825453-chatgpt-release-notes
